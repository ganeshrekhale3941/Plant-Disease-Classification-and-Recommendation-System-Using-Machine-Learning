# 🌿 Plant Disease Classification & Recommendation System

## 📌 Overview
This project is a Machine Learning-based web application that detects plant diseases and provides recommendations for treatment. It helps in early disease identification to improve crop health and productivity.

## 🚀 Features
- Disease prediction using trained ML model
- Recommendation system for treatment
- Flask-based web application
- User-friendly interface

## 🛠️ Tech Stack
- Python
- Pandas, NumPy, Scikit-learn
- Flask

## 📊 Model Details
- Algorithm: Random Forest
- Includes Feature Engineering & Model Evaluation

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies:
   pip install -r requirements.txt  
3. Run the app:
   python app.py  
4. Open browser:
   http://127.0.0.1:5000/
